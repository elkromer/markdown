# C++ Conventions

## Naming 

Pointers are generally named with a "p" prefix. References are generally named with a "r" prefix:

```
Widget *pw;

class Airplane;
Airplane *pa;

class GameCharacter;
GameCharacter *pgc;

Airplane& ra;
GameCharacter& rgc;
```
